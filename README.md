# Quantum Catalyst Discovery and Screening Platform

## Overview
An interactive platform for catalyst discovery using qubit emulation, quantum machine learning, and VQE simulation.

### Features
- Reaction selection
- Catalyst input & validation
- 3D molecule viewer
- Quantum simulation (VQE)
- Reaction pathway analysis
- Catalyst scoring
- Energy graphs & reaction animation

### Tech Stack
- Python, Streamlit, RDKit, Py3Dmol, Qiskit, Qiskit Nature, Matplotlib

### Architecture
User Input → Validation → Molecule Generation → 3D Visualization → Reaction Pathway → Quantum Simulation (VQE) → Energy Analysis → Catalyst Scoring → Dashboard
